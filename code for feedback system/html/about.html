About

This feedback management system helps citizens to establish two-way communication with the public and improve police working through the analysis of feedback received. This system is paperless and allows citizens visiting the police station to scan a QR code and mark their feedback responses. The response database is then utilized to analyze and take corrective steps for improvement in the police-public interface.The identity of the user is not disclosed.

Vission

The vision of the Police Feedback System for Rajasthan Police is to create a secure cyber space and improve the service delivery system of the police. This system is part of the broader initiative by Rajasthan Police called RACCAM (Rajasthan Police Cyber Crime Awareness Mission), which aims to deal with digital threats and cybercrime.

Mission

The mission of the Police Feedback System for Rajasthan Police is to improve transparency and efficiency by providing an accessible, real-time platform for community members to share their experiences and feedback on police interactions.
